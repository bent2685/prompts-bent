### Spec 进化指令

```
请作为资深架构师，为我产出这份功能的 **Technical Spec**。
要求涵盖：
- **边界条件**：明确这个功能『不做』什么。
- **数据模型**：定义核心 Entity 及其关系。
- **接口定义**：列出所有 API 的 Input/Output 契约。
- **异常处理**：定义所有可能的 Fail Cases。
- **性能/安全约束**：如 QPS 要求或权限校验逻辑。
在 Spec 获得我确认之前，不要生成任何实现代码。”
```
