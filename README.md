# prompts-bent

bent 个人向的提示词（Prompt）合集，专注于与 AI 编码助手（Claude Code / Cursor / Copilot 等）协作时的高频工作流场景，覆盖需求梳理、架构审计、任务开闭环、反复询问治理、异常处理、记忆与决策归档等。

## 为什么做这个

- **复用成本低**：把用过且验证有效的 Prompt 沉淀下来，遇到相同场景直接抄。
- **工作流闭环**：按「任务开始 → 开发协作 → 任务结束 → 归档复盘」串成一条链路。
- **抗遗忘**：记忆、决策、归档相关 Prompt 让下一次会话能够无缝接续。

## 使用方式

1. 按需打开对应场景的 Markdown 文件。
2. 复制代码块中的 Prompt，替换其中的占位符（如 `[业务/功能描述]`）。
3. 粘贴到你使用的 AI 编码助手中即可。

提示词设计偏向直接可用，如果某一条不适用你的项目风格，欢迎 Fork 改造。
